## Características principales

| Módulo | Funcionalidad |
|---:|---|
| Gestión de productos | CRUD, historial de cambios, búsqueda, filtros, paginación, ordenación y alertas de stock bajo. |
| Categorías | CRUD y asociación a productos. |
| Usuarios & Roles | Registro, login, gestión de roles, perfiles. |
| Pedidos (Orders) | Creación, listado por usuario, detalle y gestión de estado. |
| Detalles de pedidos | Líneas de pedido (cantidad, precio unitario). |
| Facturación (Invoice) | Generación de facturas por pedido. |
| Pagos (Pay) | Registro y consulta de estados de pago (simulados). |
| Dashboard | Métricas y estadísticas administrativas. |
| Seguridad | Spring Security + JWT; protección por roles. |
| Modo prueba | DB separada + TestDataLoader para pre-cargar datos. |
