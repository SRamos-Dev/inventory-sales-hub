# Introducción:
En un mundo donde la digitalización es clave para la eficiencia operativa, contar con un sistema de gestión de inventarios y ventas bien estructurado es fundamental para cualquier negocio.

Este manual describe el proceso de diseño y desarrollo de una aplicación web y móvil creada con Spring Boot y Java, cuyo objetivo es:

Optimizar la administración de productos

Facilitar la compra para el usuario

Mejorar la experiencia general mediante una plataforma intuitiva
