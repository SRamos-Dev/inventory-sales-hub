## Resumen del proyecto

**Inventory Sales Hub** es una aplicación backend diseñada para simular y gestionar un entorno de comercio digital: productos, inventario, pedidos, facturas, pagos, usuarios y roles (ADMIN / USER). Está pensada para ser consumida por frontends (web o móvil) y para servir como base de un producto mínimo viable, laboratorio de pruebas o ejemplo educativo/empresarial.
